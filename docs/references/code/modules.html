Code Reference

Initialization Code

marmot.marmot_h5_formatter Main formatting source code to format modelling results for plotting.
marmot.marmot_plot_main Main plotting source code, creates output figures and data-tables.
marmot.meta_data Retrieve metadata from PLEXOS production cost modelling results.

Config Package

marmot.config.mconfig This module creates the default config.yml file that is used by Marmot.

Plotting Modules Package

marmot.plottingmodules.curtailment Device curtailment plots.
marmot.plottingmodules.emissions Generator emissions plots.
marmot.plottingmodules.generation_stack Timeseries generation stacked area plots.
marmot.plottingmodules.generation_unstack Timeseries generation line plots.
marmot.plottingmodules.hydro Hydro generator plots.
marmot.plottingmodules.prices Locational price analysis plots.
marmot.plottingmodules.production_cost System operating cost plots.
marmot.plottingmodules.ramping Generator start and ramping plots.
marmot.plottingmodules.reserves Generator reserve plots.
marmot.plottingmodules.sensitivities System sensitivity plots

marmot.plottingmodules.utilization_factor Note! Module is currently not production ready.

Plot Utility Package

marmot.plottingmodules.plotutils.plot_data_helper Methods used to assist with the creation of Marmot plots.

marmot.plottingmodules.plotutils.plot_library Plot library for creating regularly used plot types.
